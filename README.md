# ecommerce-data-analysis
# 电商数据分析项目
电商零售订单数据分析实战项目，从原始订单数据出发，完成数据清洗、指标统计、用户分层、留存分析与可视化的全流程分析，沉淀可复用的电商数据分析方法与业务报表。

---

## 📋 项目背景
电商行业用户运营与增长决策高度依赖数据支撑，本项目基于真实零售订单数据，搭建完整的数据分析闭环，为用户分层运营、留存优化、业务监控提供数据依据。

## 🛠️ 技术栈
- 数据处理：Python / Pandas / Numpy
- 可视化：Matplotlib / Seaborn
- 报表输出：Excel
- 分析模型：RFM用户分层、同期群（Cohort）留存分析
